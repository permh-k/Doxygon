# 1. はじめに

## 1.1 Doxygonとは

Doxygon は、ソースコードに記述したコメントからドキュメントを生成するツールです。

Doxygon コメントでは、Doxygen コマンドと AsciiDoc の記法を組み合わせて記述できます。

生成されたドキュメントは AsciiDoc 形式で出力されるので、HTML や PDF などへ変換することができます。

Doxygon は以下の言語に対応しています。

- VBA
- Python
- C
- C++
- Java
- JavaScript
- TypeScript

---

## 1.2 Doxygon の特徴

Doxygon は、Doxygen のコメント記法と AsciiDoc の表現力を組み合わせることで、保守性と可読性の高いドキュメント作成を目指しています。

主な特徴は次のとおりです。

- Doxygen コマンドを利用したコメント記述
- 複数言語に対応した統一的なコメント記法
- AsciiDoc 記法による豊かなドキュメント表現
- PlantUML による図の埋め込み
- ソースコードと設計書の一元管理

# 2. Doxygon コメント

## 2.1 Doxygon コメントとは

Doxygon は、ソースコード中に記述された **Doxygon コメント**を解析し、ドキュメントを生成します。

Doxygon コメントには、**Doxygon コマンド**および **AsciiDoc の記法**を記述できます。

本ガイドでは、`@` で始まる命令を **Doxygon コマンド**と呼びます。

Doxygon コマンドには、Doxygen 由来のコマンドと Doxygon 独自のコマンドがあります。

各コマンドの詳細については、第3章で説明します。

---

## 2.2 コメントの種類

Doxygon コメントには、複数行にまたがる**ブロックコメント**とコードと同じ行に記述する**インラインコメント**があります。

### ブロックコメント

| 言語 | 開始記号 | 終端記号 |
|------|---------|---------|
| Python　 | `"""!` | `"""` |
| C系言語　 | `/**!` | `*/` |
| VBA　 | `'/*!` | `'*/` |

※ C系言語には、C、C++、Java、JavaScript、TypeScript が含まれます。

### インラインコメント

| 言語 | 開始記号 | 終端記号 |
|------|---------|---------|
| Python | `"""!<` | -
| C系言語　 | `/**!<` | `*/` |
| VBA　 | `'/**!<` | -

---

## 2.3 ブロックコメント

ブロックコメントは、ファイル、関数、クラスなどの説明を記述するために使用します。

各言語のコメントブロック内に Doxygon コマンドおよび AsciiDoc の記法を記述します。

### Python

```python
"""!
 @file sample.py サンプルプログラム
 @brief これはサンプルプログラムの例です。
"""
```

### C / C++

```c
/**!
 @file sample.c サンプルプログラム
 @brief これはサンプルプログラムの例です。
*/
```

### VBA

```vb
'/*!
' @file sample.bas サンプルプログラム
' @brief これはサンプルプログラムの例です。
'*/
```

その他の対応言語についても、コメントブロックの記述方法は同様です。

---

## 2.4 インラインコメント

インラインコメントは、変数、構造体メンバー、列挙値などの説明を記述するために使用します。

### C / C++

```c
int count;      /**!< 現在の件数 */
```

### VBA

```vb
Dim count As Integer    '/**!< 現在の件数
```

インラインコメント内にも、Doxygon コマンドおよび AsciiDoc の記法を記述できます。
